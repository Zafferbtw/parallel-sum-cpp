# parallel-sum-cpp
Многопоточный калькулятор суммы чисел на C++

Простая многопоточная программа на C++ для вычисления суммы чисел от 1 до N с использованием нескольких потоков.

## Описание

Программа демонстрирует основы многопоточного программирования в C++:
- Разделение задачи на подзадачи
- Создание и управление потоками
- Синхронизация доступа к общим данным с помощью мьютексов
- Распределение нагрузки между потоками

## Особенности

- **Многопоточность**: Использует 4 потока для параллельных вычислений
- **Синхронизация**: Потокобезопасный доступ к общей сумме через `std::mutex`
- **Валидация**: Проверка корректности входных данных
- **Верификация**: Сравнение результата с формулой суммы арифметической прогрессии
- **Гибкость**: Поддержка аргументов командной строки и интерактивного режима

## Требования

- Компилятор C++ с поддержкой C++11 или выше
- **Linux/macOS**: g++, clang++
- **Windows**: MinGW, Visual Studio 2015+

## Быстрый старт

1. Клонирование репозитория
git clone https://github.com/ваш-username/parallel-sum-cpp.git
cd parallel-sum-cpp

2. Компиляция
Linux/macOS:
g++ -std=c++11 -pthread parallel_sum.cpp -o parallel_sum

Windows (MinGW):
g++ -std=c++11 parallel_sum.cpp -o parallel_sum.exe

Windows (Visual Studio):
1.	Создайте новый проект Console Application
2.	Добавьте файл ConsoleApplication84.cpp
3.	Установите стандарт C++11 в настройках проекта
4.	Скомпилируйте проект

Архитектура
Основные компоненты:
1. Класс ParallelSumCalculator
•	Управляет многопоточными вычислениями
•	Содержит общую сумму и мьютекс для синхронизации
2. Метод calculatePartialSum()
•	Вычисляет сумму в заданном диапазоне
•	Безопасно добавляет результат к общей сумме
3. Метод calculateTotalSum()
•	Разбивает диапазон на части
•	Создает и запускает потоки
•	Координирует выполнение


4. Функция getInputNumber()
•	Обрабатывает пользовательский ввод
•	Проверяет корректность данных

## Изменение количества потоков
•	По умолчанию используется 4 потока. Чтобы изменить количество:
•	Откройте файл parallel_sum.cpp
•	Найдите строку: const int DEFAULT_THREADS = 4;
•	Измените значение на нужное количество потоков
